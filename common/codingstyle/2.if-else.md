## coding style

### less if-else

1. use return

   - before

   ```JAVA
   public void op(int input) {
       if(input > 5) {
           // do
       } else {
           // do else
       }
   }
   ```

   - after

   ```JAVA
   public void op(int input) {
       if(input > 5) {
           // do
           return;
       }

       // do else
   }
   ```

2. use Map

   - before

   ```JAVA
   if (t == 1) {
       type = "name";
   } else if (t == 2) {
       type = "id";
   } else if (t == 3) {
       type = "mobile";
   }
   ```

   - after

   ```JAVA
   Map<Integer, String> typeMap = new HashMap<>();
   typeMap.put(1, "name");
   typeMap.put(2, "id");
   typeMap.put(3, "mobile");

   type = typeMap.get(ty);
   ```

3. use `?:`

   - before

   ```JAVA
   Integer score = 81;
   if (score > 80) {
       score = 100;
   } else {
       score = 60;
   }
   ```

   - after

   ```JAVA
   score = score > 80 ? 100 : 60;
   ```

4. enum

   - before

   ```JAVA
   Integer typeId = 0;
   String type = "Name";
   if ("Name".equals(type)) {
       typeId = 1;
   } else if ("Age".equals(type)) {
       typeId = 2;
   } else if ("Address".equals(type)) {
       typeId = 3;
   }
   ```

   - after

   ```JAVA
   public enum TypeEnum {
       Name(1), Age(2), Address(3);

       public Integer typeId;

       TypeEnum(Integer typeId) {
           this.typeId = typeId;
       }
   }

   typeId = TypeEnum.valueOf("Name").typeId;
   ```

5. Optional: `jdk-9`

   - before

   ```JAVA
   String str = "java";
   if (str == null) {
       System.out.println("Null");
   } else {
       System.out.println(str);
   }
   ```

   - after

   ```JAVA
   Optional<String> opt = Optional.of("java");
   opt.ifPresentOrElse(v -> System.out.println(v), () -> System.out.println("Null"));
   ```

6. OOP

   - before

   ```JAVA
   Integer typeId = 0;
   String type = "Name";
   if ("Name".equals(type)) {
       typeId = 1;
   } else if ("Age".equals(type)) {
       typeId = 2;
   } else if ("Address".equals(type)) {
       typeId = 3;
   }
   ```

   - after

   ```JAVA
   public interface IType {
       public Integer getType();
   }

   public class Name implements IType {
       @Override
       public Integer getType() {
           return 1;
       }
   }

   public class Age implements IType {
       @Override
       public Integer getType() {
           return 2;
       }
   }

   public class Address implements IType {
       @Override
       public Integer getType() {
           return 3;
       }
   }

   IType itype = (IType) Class.forName("com.example." + type).newInstance();
   Integer typeId = itype.getType();
   ```

7. value assign: **also can be optimize using enum**

   - before

   ```JAVA
   public String judge(String roleName ) {
       String result = StrUtils.Empty;

       if(roleName.equals("ROLE_ROOT_ADMIN")) {
           result = "ROLE_ROOT_ADMIN: " + "has AAA permission";
       } else if(roleName.equals("ROLE_ORDER_ADMIN")) {
           result = "ROLE_ORDER_ADMIN: " + "has BBB permission";
       } else if(roleName.equals("ROLE_NORMAL")) {
           result = "ROLE_NORMAL: " + "has CCC permission";
       } else {
           result = "unknown";
       }

       return result;
   }
   ```

   - after: simple

   ```JAVA
   public String judge(String roleName ) {

       if(roleName.equals("ROLE_ROOT_ADMIN")) {
          return "ROLE_ROOT_ADMIN: " + "has AAA permission";
       }

       if(roleName.equals("ROLE_ORDER_ADMIN")) {
           return "ROLE_ORDER_ADMIN: " + "has BBB permission";
       }

       if(roleName.equals("ROLE_NORMAL")) {
           return "ROLE_NORMAL: " + "has CCC permission";
       }

       return "unknown";
   }
   ```

   - after: enum

   ```JAVA
   public interface RoleOperation {
      default String op(){
          return "unknown";
      }
   }

   public enum RoleEnum implements RoleOperation {
       ROLE_ROOT_ADMIN {
           @Override
            public String op() {
                return "ROLE_ROOT_ADMIN:" + " has AAA permission";
            }
       },
       ROLE_ORDER_ADMIN  {
           @Override
            public String op() {
                return "ROLE_ORDER_ADMIN:" + " has BBB permission";
            }
       },
       ROLE_NORMAL   {
           @Override
            public String op() {
                return "ROLE_NORMAL :" + " has CCC permission";
            }
       };
   }

   public String judge(String roleName ) {
       return RoleEnum.valueOf(roleName).op();
   }
   ```

9. strategy: define StrategyContext

   - before: `2#before`
   - after

   ```JAVA
   public class RoleContext {
       private RoleOperation roleOperation;

       public RoleContext(RoleOperation roleOperation) {
           this.roleOperation = roleOperation;
       }

       public String execute() {
           return roleOperation.op();
       }
   }

   public class JudgeRole {
       public String judge(RoleOperation roleOperation) {
           RoleContext context = new RoleContext(roleOperation);

           return context.execute();
       }
   }

   // not suitable for this case
   ```
