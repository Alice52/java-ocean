## shotcut

|     shotcut      | function |
| :--------------: | :------: |
|     CTRL + =     |   下标   |
| CTRL + SHIFT + = |   下标   |
|      ctrl+3      |   箭头   |
|      ctrl+6      |   直线   |
|      ctrl+g      |   组合   |
|   ctrl+shift+u   | 取消组合 |
|       F12        |   保存   |

## custom Mould

1. 选择更多形状

3. 将 模具放入左边
