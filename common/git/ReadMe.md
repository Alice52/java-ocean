## usage

1. push code to gitee and github[origin]

   ```shell
    # config
    git remote add gitee  GITEE_REPO_URL

    # should push separately
    # push to github
    git push origin BRANCH_NAME

    # push to gitee
    git push gitee BRANCH_NAME
   ```
